# TechSummit 2026 Event Talks App

A single-page application to view and search the schedule of a 1-day technical event.

## Features
- Single track schedule with 6 talks.
- 1-hour talks with 10-minute transitions.
- Lunch break included.
- Real-time search by category.
- Modern dark-themed tech UI.

## Tech Stack
- **Backend:** Node.js, Express.
- **Frontend:** HTML, CSS, Vanilla JavaScript.

## Getting Started
1. Clone the repository.
2. Run `npm install`.
3. Run `node server.js`.
4. Open `http://localhost:3333`.
